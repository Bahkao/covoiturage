Administration


Ici vous avez la possibilité de gérer les modules suivants :
  • : Ce module vous permet d'ajouter ou supprimer un utilisateur. Vous pouvez également modifier ses informations et lui ajouter/supprimer des droits. Pour ce faire, vous pouvez les ajouter manuellement, ou alors ajouter cette personne dans un groupe possédant les droits voulus.

  • : Ce module vous permet d'ajouter, modifier et supprimer des avis. Il est toutefois conseillé d'utiliser les fonctionnalités internes du site.

  • : Ce module permet de créer des groupes d'utilisateurs et de leur assigner des permissions d'une manière plus simple.

  • : Ce module vous permet d'ajouter, modifier et supprimer un lieu d'arrivée qui sera utilisé lors des fonctions de recherche de covoiturage. Attention toutefois qu'une modification d'un lieu d'arrivée engendre la suppression de tous les enregistrements en relation avec ce lieu tels que les trajets, demande et même les covoiturages déjà établis.

  • : Cette page vous permet de modifier la page de contact du site. Elle est sensée contenir des informations relative à votre organisation. La modification de cette page se fait via le langage HTML.

  • : Ce formulaire vous permet de modifier le mot de passe demandé lors de la création d'un compte utilisateur. En cas de changement, veillez à transmettre le nouveau mot de passe à votre organisation et ce afin de prévenir les membres de celle-ci.

  • : Ce module vous permet d'accéder à des fonctionnalités d'administration plus avancées. Il est cependant déconseiller d'effectuer trop de modification par cet outil. Pour plus d'informations veuillez contacter le concepteur du système.
